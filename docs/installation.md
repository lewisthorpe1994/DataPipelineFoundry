# Installation

## Download and extract

1.  Grab the archive for your OS/architecture from the [GitHub Releases page](https://github.com/lewisthorpe1994/DataPipelineFoundry/releases).
2.  Extract it to a directory on disk, for example:
    -   macOS/Linux: `tar -xzf dpf-v0.1.0-linux-x86_64.tar.gz`
    -   Windows: unzip `dpf-v0.1.0-windows-x86_64.zip` with Explorer or PowerShell.
3.  (macOS/Linux only) mark the binary as executable if needed: `chmod +x dpf`.
